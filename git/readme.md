## git

### SVN 集中式版本控制系统

```
(代码仓库)
SVN Server : 维护了代码的所有的版本
|                  |                   |
|                  |                   |
client 1           client 2          client 3
(SVN client)     (SVN client)     (SVN client)

通过网络通信连接，所有的增删改都是在SVN server中发生，SVN只是做请求

一般在局域网上进行共同开发
```
### git分布式版本控制系统

```

(代码版本托管系统 e.g. github) 
远程仓库


(本地仓库)  (本地仓库)     (本地仓库)
git1         git2         git3

// 每一个都是一个独立的版本控制系统，工作在一个结点，结点和结点之间不需要构成一个网络
// 单独就可以工作
// 然后提交到远程仓库，进行合并
```
