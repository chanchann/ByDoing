## 开机

1. 刚开机CPU处于实模式 (和保护模式对应，实模式寻址CS:IP,CS左移4位+IP) todo: 保护模式寻址

2. 开机时，CS = 0xFFFF;IP = 0x0000 (段:偏移)

3. 寻址0xFFFF0(ROM BIOS映射区)

4. 检查RAM, 键盘，显示器，软硬磁盘

5. 将磁盘0磁道0扇区读一个扇区(512个字节)到0x7c00处
